# C Extension Collection Fork


I'm generally a fan of the C feature addition process. While C++ accepts almost any proposal thrown its way, C tends to add only the things that stick, while also prioritizing backwards compatibility and sticking with a Spartan language design philosophy.

However, there are a number of features that either aren't being added, or need to wait a long time before the next version. So I decided why not just implement them myself. Maybe, for the features that align with the C standard well, I will contribute to the next release (c2y).

But for now, they will live in this repo, as CXC (C eXtension Collection). CXC is a superset of C23.

## Features
* Native `defer` keyword.
* Native `noreturn` keyword.
